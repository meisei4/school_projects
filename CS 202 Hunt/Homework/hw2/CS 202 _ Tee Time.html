HTML Documents: Tables, Forms and all kinds of stuff

Overview

You are starting an online customized t-shirt store. Your company is named Tee Time. Write a website for this company that consists of three HTML files as described below.

Page Descriptions

Global Requirements

You must create several html documents where each of these documents is named and described in the subsections that follow. Ensure that each file is located in a single folder named homework2. Each of your documents must contain a header and a footer. Each document will contain the same header and footer. Do not include full paths in any link. Use relative paths only.

Header

The header is a major navigation element that must include links that are displayed as

  1. Home (linking to the index.html file)
  2. About (linking to the about.html file)
  3. Order (linking to the order.html file)

The header must have a logo. The logo an image that identifies your store. internet. Make sure that it is appropriate for public viewing and free of copyright. I recommend using Wikimedia or Open Clip Art content.

In addition to the logo, the header must include the name of your store.

Footer

The footer contains a major navigation element that must include links that are displayed as

  1. About (linking to the about.html file)
  2. Order (linking to the order.html file)

The contact information for your company must be included. This contact information must include a street address and an email contact. The web page is presumed to be authored by the hobbyist group (most likely authored on their behalf by a software contractor).

The footer must also include a copyright notice.

index.html

Create a document named index.html. This document is the entry point for the web site. The document must have an outline shown in Figure 1.

Each best-in-the-business item is a short paragraph of content that describes why your business is superior with respect to selection, quality and ease of mind. Make something up that sounds like typical marketing jargon.

Each occassion is a short paragraph that describes how customized t-shirts would be such a great thing to have. Make something up that sounds like typical marketing jargon.

Each customer review is a short paragraph that praises your store. Each review should have the name, date, and a graphical star-rating of the review. Each rating is on a 1-5 star scale.

about.html

This document contains information about your store. You must include ficiontal information to satisfy the requirements of this page. The outline of the about page is given in Figure 4.

The history must be at least two paragraphs that describe how the store was formed.

The philosophy must be at least one paragraph that desribes your business philosophy.

The people must describe at least two individuals by name and others in general. You might list the president-and-founder or the vice-president and then mention the others as a whole.

Requirements and Grading

  1. Email a single zip file of your homework1 folder. Do not email a file in any other format (rar, gz, etc...).
  2. Ensure that each image you use is either a) not protected by copyright or b) you own the copyright.
  3. Your grade will be based, primarily, on the following factors:
    1. The correct use and structure of the elements in the document.
    2. The layout of the documents source code (this is not related to how the document appears in a browser when rendered).
    3. The validity. Check validity with the W3C validator
  4. You will not be graded on the visual appearance of the documents when rendered in a web browser. Do not include any styling directives anywhere in your HTML code.
